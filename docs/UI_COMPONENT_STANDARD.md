# UI_COMPONENT_STANDARD

## Shared Components

DS-011 Reusable Intelligence: recommendation, matchup, roster, recruiting, and playbook intelligence should be resolved once through shared helper layers and rendered through shared components.

Cards support compact and expanded states, title/subtitle, badges, portraits/media, metric rows, stat sections, source status, actions, priority variants, and size variants.

The bottom navigation is fixed and floating. The coordinator header is sticky and compacts after scroll without hiding content.

Default views stay compact; long football evidence belongs in accordions, detail rows, or full breakdown panels.
